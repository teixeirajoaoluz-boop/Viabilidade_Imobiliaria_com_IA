# Viabilidade_Imobiliaria_com_IA
# Vifrancon - Plataforma de Viabilidade Financeira Imobiliária

> Plataforma web para análise de viabilidade financeira de empreendimentos imobiliários, desenvolvida com HTML5, CSS3 e JavaScript puro, utilizando Engenharia de Prompts e Inteligência Artificial aplicada ao desenvolvimento de software financeiro.

---

# Sobre o Projeto

O **Vifrancon** é uma aplicação desenvolvida para simular e analisar a viabilidade econômico-financeira de empreendimentos de incorporação imobiliária.

O objetivo do projeto foi unir conhecimentos de **Contabilidade, Controladoria, FP&A, Corporate Finance e Engenharia Financeira** com **Inteligência Artificial Generativa**, demonstrando como modelos de IA podem acelerar o desenvolvimento de aplicações financeiras robustas.

Mais do que um estudo técnico, o projeto representa uma demonstração prática de como profissionais da área financeira podem utilizar Engenharia de Prompts para construir soluções reais.

---

# Objetivos

- Automatizar análises de viabilidade financeira.
- Simular cenários de investimentos imobiliários.
- Consolidar indicadores financeiros em um dashboard executivo.
- Apoiar processos de tomada de decisão.
- Demonstrar aplicações práticas de IA em Finanças.
- Compor um portfólio voltado para FP&A, Controladoria e Inteligência Financeira.

Este projeto foi desenvolvido utilizando um processo estruturado de **Engenharia de Prompts**, no qual foram definidos:

- Arquitetura da aplicação
- Modelagem financeira
- Indicadores
- Interface
- Regras de negócio
- Estrutura do código
- Critérios de qualidade

A Inteligência Artificial foi utilizada como ferramenta de desenvolvimento para acelerar a implementação, enquanto toda a definição da solução, refinamento dos requisitos, validação funcional e direcionamento do projeto foram conduzidos durante o processo de desenvolvimento.
